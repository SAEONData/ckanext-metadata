{{ _('What are attribute mappings?') }}

{% trans %}

An attribute mapping is a one-to-one mapping from a metadata JSON element to a metadata record attribute. It applies whenever a metadata record is created or updated, causing the value of the specified JSON element to be copied into the mapped attribute.

If marked as a key attribute, the value of the JSON element is used to determine whether to create a new metadata record in the database or to update an existing one.

Note: if multiple key attributes are defined, an existing record must be matched on all keys; a partial match is an error.
